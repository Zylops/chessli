## Basic Usage
To get help about `chessli`s commands, open your console and type:
```console
chessli --help
```
The same works for subcommands, e.g., :
```console
chessli games --help
```
You can find an overview of all availabe cli-commands [here](https://pwenker.com/chessli/cli/) in
the docs.
